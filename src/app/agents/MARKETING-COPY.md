## PROBLEM SECTION — "Here's what breaks"

**Header:** The mega-agent problem

**Copy:**

Most people build one AI agent and pile on skills — calendar, email, Notion, content, research, code, outreach. All in one.

It looks powerful. It fails in practice.

The more skills you stack, the worse it performs. Context gets clouded. It fires the wrong integration at the wrong time. The personality fragments. You end up managing it more than it helps you.

We've seen it. We've tested it. The research confirms it: **above 7–10 skills, agent dependability drops sharply.**

So we stopped building mega-agents.

## SOLUTION SECTION — "How we build"

**Header:** Narrow agents. Clear goals. Actual results.

**Copy:**

We build focused agents — each one owning a specific function with 7–10 skills maximum.

A content agent that knows your voice, your platforms, and your calendar. Nothing else.
A research agent that scans your sources, extracts signal, and files it where you need it. Nothing else.
A newsletter agent that reads your journal, picks the best story, and drafts the send. Nothing else.

Each one has a measurable goal. Pass or fail. Which means you can improve them, clone them, and trust them.

**The result:** a team of AI agents that works like a team of specialists — not one overwhelmed generalist.

## HOW IT WORKS — 3-STEP SECTION

**Step 1 — Map your real workflow**
We don't start with tools. We start with: what does your week actually look like? Where does work pile up? What are you doing on repeat that a focused agent could own?

**Step 2 — Define agent scope + goals**
Each agent gets a job description: specific goals (KPIs), specific skills (tools + integrations), and a personality aligned to its function. No bloat.

**Step 3 — Build, test, iterate**
We build on OpenClaw — agents that run on a real computer, not in the cloud on-demand. Persistent memory. Scheduled heartbeats. Connected to the tools you already use: Notion, Telegram, Slack, email, calendar.

## AGENT TYPES WE BUILD

(Cards / grid section)

**Content Agent**
Knows your voice. Monitors your inputs. Drafts posts, threads, and carousels. Flags marketing-ready moments. Feeds your queue automatically.
_Goal: content output / posting cadence / brand consistency_

**Research Agent**
Tracks sources you care about. Transcribes, distills, tags, and files. Surfaces only what's worth your attention.
_Goal: signal-to-noise ratio / KB quality_

**Outreach Agent**
Monitors conversations, drafts responses, tracks leads, manages follow-up cadence.
_Goal: reply rate / pipeline velocity_

**Ops Agent**
Manages your calendar, inbox triage, meeting prep, and daily summaries. Knows your priorities. Gets out of the way when it's done.
_Goal: hours saved / decision load reduced_

**Journal / Memory Agent**
Logs everything that matters across your sessions, meetings, and builds. Shares that memory with every other agent in your stack.
_Goal: context continuity / no dropped threads_
